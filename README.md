
# MediVision AI
_A Medical Report Interpreter using OCR + HuggingFace + LangChain_

MediVision AI is a powerful yet lightweight web application built using **Python, LangChain, HuggingFace Transformers, OCR (Tesseract), and Gradio**.  
It enables users to **upload a medical report image**, extract its text using OCR, and generate a **simple, understandable explanation** of the medical findings.
